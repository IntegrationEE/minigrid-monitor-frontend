How To Upload data via Excel:
  1. Download the current tables via the Button "Download table data file" and save it locally.
  2. Open the File with Excel.
  3. Editing:
    1. You can add new data by using the current Year/Month format
    2. You cannot delete rows but make them invalid when you type 0
    3. Year/Month duplicates are not allowed
    4. blank rows are not allowed
  4. Upload Data via the Button "Upload table data file" and select "Your edited file".
